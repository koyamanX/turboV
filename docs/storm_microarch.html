Storm Microarchitecture

Microarchitecture of storm is based on Tomasulo algorithm with speculation. It is implemented in NSL(Next Synthsis Language). In addition to storm core, SoC which uses storm is also implemented. The goals of this project is to implement higher performance core compare to prior classic five stage pipelined RISC-V core rv32x_dev and supports UNIX-like operating system.
